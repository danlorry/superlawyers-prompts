# Contributed Prompts

This folder contains prompts contributed by lawyers who have tested and refined them in practice.

---

## How to contribute

1. Fork this repository
2. Add your prompt file to this folder following the format below
3. Submit a pull request with a brief description of the practice area and use case

## Required format for contributions

```markdown
## Prompt Name

**Practice area:** [Specific area of law]
**Jurisdiction notes:** [Any jurisdiction-specific considerations]
**Use when:** [Specific situation that triggers this prompt]
**Contributed by:** [Optional — your name or anonymous]
**Verified in practice:** [Yes/No — have you used this in a real matter]

[PROMPT TEXT]

**What to verify:** [Specific outputs that require verification before use]
**Known limitations:** [Where this prompt produces weaker output]
```

## Acceptance criteria

Contributed prompts must:
- Follow the CONTEXT-TASK-FORMAT-CONSTRAINTS structure
- Include explicit verification instructions
- Not encourage submission of AI-generated citations without verification
- Be tested in at least one real matter before submission

Prompts that generate unverified legal citations or omit verification steps will not be accepted.
